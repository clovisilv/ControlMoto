ABRIR ORDEM DE SERVIÇO

Abrir

Para abrir uma ordem de serviço clique no menu Operacional, depois em Abrir O.S., e na tela que aparecer clicar no botão Incluir, agora no campo Código Cliente digite o código do cliente que solicitou o serviço e aperte a tecla Enter, cofira os dados do cliente.

Para preencher os campos com às informações do Motorista, faça da mesma forma que foi feito para o cliente e confira os dados.

Agora é só digitar a data e hora de início e clicar no botão Gravar.

Localizar

Para localizar uma ordem de serviço, clique no botão Localizar e digite o número da ordem de serviço a ser procurada e clique em OK.

Atualizar

Para Atualizar é preciso fazer antes o procedimento de Localizar, e realizar as modificações desejadas e clicar no botão Atualizar.

Imprimir

Para Imprimir a ordem de serviço basta clicar no botão correspondente, caso queira imprimir uma ordem que não foi impressa na primeira vez, faça o processo de Localizar a ordem de serviço e clique no botão Imprimir, se este botão não aparecer é sinal que a ordem de serviço já está impressa.

Cancelar

Para Cancelar uma ordem de serviço faça o procedimento de Localizar, verifique se é está a ordem de serviço que quer cancelar, e para confirmar o procedimento clique no botão Cancelar.
