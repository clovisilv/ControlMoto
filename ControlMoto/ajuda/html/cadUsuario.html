CADASTRO DE USUÁRIO

Para Cadastrar um Usuário clique no menu Cadastro depois em usuário. Após a abertura da tela de usuário, clique no botão incluir e digite as informações, depois de todas informações digitadas clique no botão gravar.

Para localizar um usuário, na tela de cadastro de usuário clique no botão Localizar na tela que abrirá digite o código do cliente que deseja fazer a pesquisa e clique em OK.

Para excluir é preciso realizar antes o procedimento de Localizar, e depois de localizado o usuário, confirme se é o usuário que deseja excluir e clique no botão excluir.

Para atualizar é também &ecute; preciso fazer o procedimento de Localizar, confirmar se o usuário está correto, e realizar as modificações desejadas, e clicar no botão atualizar.
Este procedimento é utilizado para mudar as informações do usuário quando os dados de acesso expirarem.

O botão cancelar finaliza o procedimento que estiver sendo realizado (Incluir e Atualizar) no momento, sem afetar qualquer informação do usuá já cadastrado, se o usuá estiver sendo cadastrado pela primeira vez o mesmo não ser´ realizado. o.

Para sair da tela de cadastro é só clicar em Sair, ou no X que aparece no canto superior direito da tela.
