CADASTRO DE CLIENTES

Para Cadastrar um Cliente clique no menu Cadastro depois em Cliente. Após a abertura da tela de cliente, clique no botão incluir e digite as informações, depois de todas informações digitadas clique no botão gravar.

Para Localizar um cliente, na tela de cadastro de cliente clique no botão Localizar na tela que abrirá digite o código do cliente que deseja fazer a pesquisa e clique em OK.

Para excluir é preciso realizar antes o procedimento de Localizar e depois de localizado o cliente, confirme as informações e se é este o cliente que deseja apagar e clique no botão excluir.

Para atualizar é preciso fazer antes o procedimento de Localizar, confirmar se o cliente está correto, e realizar as modificações desejadas, e clicar no botão atualizar.

O botão cancelar finaliza o procedimento que estiver sendo realizado (Incluir e Atualizar) no momento, sem afetar qualquer informação do cliente já cadastrado ou que estiver sendo cadastrado.

Para sair da tela de cadastro de cliente é só clicar em Sair, ou no X que aparece no canto superior da mesma tela.
